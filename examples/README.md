# Examples

Sample FiveM resources that demonstrate s4-doctor integration. These are **not** loaded automatically — copy the folder you need into your server's `resources` directory.

| Resource | Description |
|----------|-------------|
| [doctor-test](doctor-test/) | Vehicle spawn, plate, warp-into-seat — full agent test flow |

Each example includes its own `README.md` with `server.cfg` steps and `POST /execute` payloads.
