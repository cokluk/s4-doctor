# API client scripts

Optional Node.js and Python helpers for calling the s4-doctor HTTP API (`http://127.0.0.1:4789`).

| File | Run |
|------|-----|
| `example-node.js` | `node docs/scripts/example-node.js` (Node 18+) |
| `example-python.py` | `python docs/scripts/example-python.py` |

Environment variable: `S4_DOCTOR_URL` (default `http://127.0.0.1:4789`).

These are documentation samples — the runtime API lives in `api/server.js`.
